========================================================================
    Group degree centralization
========================================================================

Loads a graph and computes group degree centralization measure.

/////////////////////////////////////////////////////////////////////////////
Parameters:

   -i:Input graph (tab separated list of edges) (default:'./Data/facebook.txt')
   -o:Output file name (default:'./Output/node_centrality.txt')

/////////////////////////////////////////////////////////////////////////////
Usage:

gcentralization.exe -i:graph.txt -o:out.txt
